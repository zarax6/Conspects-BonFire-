## Основные определения

- Оперативная память (ОЗУ или RAM) - это энергозависимая, быстрая и центральная часть, которая служит как временное хранилище данных.
- Благодаря своей структуре обеспечивает высокую скорость обмена данными с центральным процессором.

---

## Основные характеристики

- Энергозависимость, т.е. данные не могут храниться без постоянного источника энергии.
- Произвольный доступ, т.е. к любой ячейке можно обратиться напрямую, без обхода остальных ячеек.
- Ячейка памяти позволяет хранить 1 бит информации благодаря связке транзистора и конденсатора.
- Матрица памяти - двумерная структура (массив, матрица), состоящая из ячеек памяти.
- Адресация - каждая ячейка памяти хранится в матрице по указанному адресу (индексу).

---

## Метрики ОЗУ

- CAS latency (CL): Задержка между подачей адреса столбца и началом передачи данных.
- RAS to CAS delay (tRCD): Время между активацией строки и столбца.
- ROW Precharge Time (tRP): Время подготовки строки к следующему доступу.
- Cycle Time (tRAS): Минимальное время удержания строки в активном режиме.

Эти метрики влияют на общую скорость работы памяти.

# Реализация базовых структур в Python
### Для работы понадобятся:

- Функции

## Функции

Функция - контейнер, имеющий собственное имя, а также входные и выходные параметры. Иными словами, это обособленный блок кода.

В Python объявляется при помощи **ключевого слова def name функции**.

- Блок кода
- Возврат выходных данных

Функция может ничего не принимать и не возвращать.
## Массивы (Списки в Python)

**Массив** - структура данных, хранящая значения, каждое из которых имеет свой индекс. Массив может быть многомерным (2-х мерные, 3-х мерные массивы). Доступ к элементам массива осуществляется при помощи квадратных скобок, внутри которых указываются те или иные значения через запятую (для многомерных). Чтобы обратиться и получить элемент массива, нужно указать его имя и индекс элемента в квадратных скобках. Индексация в массивах начинается с нуля (0).
