# Docker
Docker — это платформа контейнеризации, которая позволяет упаковывать приложение вместе со всеми его зависимостями в единый переносимый объект — контейнер.
Контейнеры работают поверх ядра операционной системы (обычно Linux) и используют механизмы изоляции (namespaces, cgroups), благодаря чему:
1. процессы внутри контейнера изолированы от системы и других контейнеров
2. ресурсы (CPU, память, диск) можно ограничивать
3. окружение полностью контролируется
------------------------------------------------------------------------
Преимущество: Docker не виртуализирует железо, как это делает VM. Он использует ядро хост-системы, поэтому контейнеры:
- запускаются быстрее
- занимают меньше места
- проще масштабируются
# Из чего состоит
Docker состоит из нескольких частей:
1. Docker Engine — основной демон (daemon), который управляет контейнерами
2. Docker CLI — интерфейс командной строки
3. Docker API — для автоматизации и интеграций
