
# Docker
**Инструмент для контейнеризации, который упаковывает приложение с зависимостями в легкий изолированный контейнер. Он изолирует процессы на уровне ядра и работает на одном хосте. Его задача увеличить совместимость, упростить деплой и воспроизводимость**

#  Deploy
**Процесс развёртывания приложения на хостинге, для обеспечения внешнего доступа**

# Kubernetes
**Инструмент для управления контейнерами обеспечивающий высокую эффективность и отказоустойчивость развёрнутых систем.**
