# Введение

Структура данных это программная единица позволяющая хранить и обрабатывать однотипные и/или логические данные для добавления, для взаимодействиями с данными структура данных предоставляет некоторый набор функций составляющий ее интерфейс

### Связный список

Базовая динамическая СД состоящая из узлов, каждый из который содержит значение и ссылку на следующий узел.
Двойной связный список это список который узлы содержать ссылки еще и на предыдущий элемент.

###  Деревья

Структура данных расширяющаяся идею связанного списка. Однако в отличие от них, они могут иметь более 1 связанного с ними узла. Узлы дерева могут иметь любое количество дочерних узлов. Благодаря чему мы видим их в виде гибкой схемы ветвления.

### Графы

При помощи графов можно расширить область применения и ослабить строго родительскую связь в них. Узлы графа не имеют явной иерархии, каждый узел может быть соединен с другим.
