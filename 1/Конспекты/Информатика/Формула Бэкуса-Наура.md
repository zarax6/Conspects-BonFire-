# Формула
Формула Бэкуса-Наура (сокр. БНФ) простой способ описывать синтаксиса ЯП или других формальных систем. БНФ помогает точно объяснить как строить (предложение) например код программы из слов. Также БНФ определяется как формальный способ записи синтаксических единиц при котором они определяются через другие единицы или через элементарные символы

# Алфавит Синтаксических единиц
1. **"::="** - разделяет левую часть правила (не терминал) от правой (определение) указывает что не терминал заменяется на указанную конструкцию
2. **"|"** (Черта) - (или альтернатива) указывает выбор между вариантами правой части правила, позволяет описывать несколько возможных форм
3. **"<>"** (угловые скобки для не терминалов) - служат для обрамления не терминалов
**эти 3 символа являются основой оригинальной БНФ**

# Термины БНФ
1. **Rule** (Правило) - формальное утверждение в БНФ записываемое как "<не терминал ::= определение>" которое описывает как не терминал может быть заменен на последовательность символов
2. **Definition** (Определение) - правое часть правила БНФ состоящая из терминалов, не терминалов и др. конструкций.
3. **Terminal** (Терминал) - конечный символ БНФ который не может быть разложен и представляет конкретные элементы ЯП. Такие как ключевые слова, операторы и литералы. 
4. **Non Terminal** (Не терминал) - Символ БНФ определяющий и представляющий категорию или абстрактную категорию языка которая может быть заменена согласно правилам.
