# Введение
Файл это часть файловой системы (минимальная единица хранения "осмысленной" информации), которая является контейнером для хранения данных. Такой способ хранения информации обеспечивает её целостность, однозначность и безопасность

# Атрибуты
Атрибуты файла могут быть обязательными и необязательными, в зависимости от файловой и операционной системы, применяемых политик управления файлами. 

К атрибутам относится имя файла, его расширения и свойства, а так же набор других уникальных атрибутов, принадлежащих конкретной ФС

Например в Windows имя файла не является обязательным атрибутом, но система не позволяет хранить несколько файлов без имени с одинаковым расширением в одном каталоге

# Форматы и расширения
Для корректной работы с информацией в файле, каждый файл должен иметь формат - некоторую категорию, к представлению которой относится информация.

Виды форматов:
1. Текстовый
2. Графический
3. Аудио
4. Архивы

Так же существуют "проприетарные" форматы которые, комбинируют в себе другие или не имеют представления для человека.

Форматы в файловой системе определяются расширением - обычно эта аббревиатура записана через точку после названия файла (Например, в Windows)
